#displayManager

Create a class/library

Class instance: displayManager(port);

Make a single, responsieve, webpage for multiple browsers, window-sizes and devices like iPad
Store the single webpage in RAM if possible
Use char array's instead of Strings
Make use of WebSockets to update the pages
Make use of LittleFS

Use data/displayManager.css as a basis for all styles

Methods:

- begin(stream &debuglOut = nullptr)
- addPage(pageName, html)
- addPage(pageName, html, widthPix, heightPix)
- setPageTitle(pageName, char * title)
- activatePage(pageName)
- removePage(pageName)
- addMenu(pageName, char * menuName)
- addMenuItem(pageName, menuName, char *itemName, callbackfunction)
- addMenuItem(pageName, menuName, char *itemName, url)
- disableMenuItem(pageName, menuName, char *itemName)
- enableMenuItem(pageName, menuName, char *itemName)
- RemoveMenu(pageName, const char *menuName)
- setMessage(char *message, duration)
- setErrorMessage(char *message, duration)

Methods explained:
- addPage(pageName):
  - The body of the page should have a <div> section for every page added. This <div> has
    default "display: None" but if activatePage() for this page is called all other <div>s in <body>
    get "display:None" and this <div> gets "display: block"
- removePage(pageName):
  - removes the <div> for this page from the <body>
